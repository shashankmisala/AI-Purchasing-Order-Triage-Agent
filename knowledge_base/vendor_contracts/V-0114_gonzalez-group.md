# Contract Terms — Gonzalez Group (V-0114)

## Partial Shipment / Backorder Clause
Per Section 8.1, Gonzalez Group is contractually permitted to fulfill orders via partial
shipment when full quantity is on backorder, billing only for quantity actually shipped on
each invoice. A "quantity received" lower than "quantity ordered" against a single invoice
is expected under this clause as long as the remaining balance is noted as backordered on
the shipment documentation, not a Quantity Mismatch requiring escalation.

## Escalation Notes
If there is no backorder note on file and the shipment is represented as complete, treat
any shortfall as a genuine Quantity Mismatch.
